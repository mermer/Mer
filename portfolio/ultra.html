Ultra Foods grocery store needed a new website where it can be easily managed by an account manager. I used designer Jodie Dudek's designs to develop a site using Wordpress with major custom post types implementation and PHP templates. The majority of the text and images can be changed via custom post types. This responsive website is built using Wordpress, PHP, HTML5, CSS3, Bootstrap, and jQuery.
