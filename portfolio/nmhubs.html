#NMatNYFW & #HolidayxNM

Neiman Marcus wanted interactive web pages for two of their most important times of the year: New York Fashion Week and the holiday season. Both web pages are a part of their blog and were created for an interactive experience with their consumers. I teamed with Aneta Kostic to develop these pages.

Schedule of all the shows, with notifications when they are live on Facebook Live or Periscope.

Showed the latest Facebook Live video by using Facebook's API and showed the latest Periscope video by using Twitter's custom widgets. This was extremely important, as this is one of the most popular ways to share the NYFW experience.

iFramed Curalate pictures inside of a slider. Curalate is a service that generates photos where you can link items and shop directly in the photo.

Added Instagram feed through a very handy Wordpress Plugin.

Linked to Neiman Marcus' Snapchat account to follow their story. The URL automatically opened snapchat if the user was on their phone or it would open a link with a picture of their unique snapchat code to add their account form there.

iFramed both Facebook and Twitter newsfeeds so the user can keep up with the latest NYFW news.

Used PHP with PODS to show all of the blog related NYFW posts. Also customized this in the Wordpress backend so that our blog writers can simply check a box for the post to be displayed in this slider.

#NMatNYFW

The purpose of this page was to share Neiman's experience at New York Fashion Week with their consumers. A lot of the NYFW experience is shared through social media. This page will be updated again for this year's Fashion week in February. I used HTML5, CSS3, CSS Animations, Javascript, and PHP with PODS implementation.

25 Days of Gifts had an advent-calendar feel where every day, a new gif would be released to show. Upon clicking the previous days in the calendar, a modal will popup with that day's gif just in case you missed it. We created this with Javascript to pull the current date so that it would automatically change each day.

#HolidayxNM

Neiman Marcus loved the NYFW page's animations so Aneta and I took this page to a whole new level. This mega-interactive page is full of moving snowflakes, falling snow, and sparkly cursor animations on click. This page was created for Neiman's consumer to find the perfect gift at NeimanMarcus.com while having a some fun. I used HTML5, CSS3, CSS Animations, Javascript, and PHP with PODS implementation.
